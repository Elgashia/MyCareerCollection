**코드 스타일(필수)**

1. componet명의 첫번째는 반드시 대문자!
2. component에서 template에 name 지정시 component 파일명과 동일해야함.
3. export를 이용해 components를 내보낼때의 route name은 반드시 파일명과 같지만 all 소문자.
